## Установка и запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка базы данных PostgreSQL
Убедитесь, что PostgreSQL запущен и создана база данных `uniquestus`:
```sql
CREATE DATABASE uniquestus;
```

Или обновите настройки в `uniquest/settings.py` или через переменные окружения:
```bash
export DB_NAME=uniquestus
export DB_USER=postgres
export DB_PASSWORD=ваш_пароль
export DB_HOST=localhost
export DB_PORT=5432
```

### 3. Применение миграций
```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. Генерация тестовых данных
```bash
python generate_kazakhstan_data.py
```

Это создаст:
- Специальности и предметы
- 15 преподавателей
- 200 студентов
- Курсы, оценки, задания
- Расписание

### 5. Создание суперпользователя (опционально)
```bash
python manage.py createsuperuser
```

### 6. Запуск сервера
```bash
python manage.py runserver
```
